Credit Card Fraud Detection

Project Goal: Detect fraudulent transactions in credit card data using machine learning.

  • Data Analysis and Visualization: Performed exploratory data analysis and visualized transaction class distribution and amount per transaction by class.
  • Data Preprocessing: Preprocessed the data by dropping non-crucial features and replacing NaN values with the mean of the column.
  • Under-Sampling: Implemented under-sampling to build a balanced dataset containing similar distribution of normal and fraudulent transactions.
  • Model Training: Trained a Logistic Regression model on the preprocessed data.
  • Model Evaluation: Achieved an accuracy of 97.34% on training data and 96.93% on test data.
